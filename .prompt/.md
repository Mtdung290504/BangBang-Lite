Đánh giá tổng thể
✅ Điểm mạnh:

Triết lý 2-primitive (Impactor + Effect) rất solid — mọi mechanic phức tạp đều quy về được, không cần thêm type.
Effect system 4-tầng (modify-stats / modify-states / on-start|interval|end / on-event) cực kỳ expressive và không redundant.
Phase system array-indexed đơn giản mà đủ mạnh — GCL 3-element cycle chứng minh.
Zero-return trick + conditional ValueResolver thay thế hoàn toàn nhu cầu condition trong action — gọn.
Sensor entity pattern, paired-effect pattern, warm-up collider — đủ cover hầu hết edge case MOBA.
